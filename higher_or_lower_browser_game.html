Higher or Lower

A Browser Game

A fun browser game featuring embedded gifs from The Office (US)

This was one of my earlier projects where I was learning about how to use Flask to create new web pages on the fly depending on what I entered.

There was no HTML in this project, everything was being done by Flask in python. I found that to be really cool.

I used the python random module to choose a random int between 0 and 9, and then flask was programmed to allow me to input integer guesses into the live server address bar.

A function then checked if the integer was correct, and depending on that, you would get a different header and gif.
