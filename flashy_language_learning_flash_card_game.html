Flashy Language Learning Flash Card Game

Learn your favourite languages using Flashy Flash Cards!

An incredibly fun project to have built, and very rewarding as well! This tests your recall on words you're learning from a new language to you.

Mine is configured to use the top 1000 most common portuguese words, as I have been learning portuguese in my free time. It references CSV files of the words where the first value is the portuguese word, and the second the english translation

Done in python, I used pandas to create a dataframe of the words from the CSV files. The purpose of this is the following: Pandas dataframes are great for manipulating large datasets. As such, as problem words are encountered while using th eapplication, they get written to a seperate CSV.

This CSV of problem words is the one that gets opened the next time the game is played. Pandas also discards words the player recalls quickly. This maximises time spent working on troublesome words and it eliminates words the player already knows.

Currently the timer is set to 3 seconds, after which time, the game will flip thew card and reveal the translation to the player. I had a lot of fun making this game and it has been very useful to me in learning portguese.
