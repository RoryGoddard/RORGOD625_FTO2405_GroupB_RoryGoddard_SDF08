Quizzler Trivia Game

A trivia game right on your PC!

A trivia game where we connect to opentdb's api to generate a set of trivia questions with true or false answers. The JSON response from opentdb is saved within a list. We then iterate over the list to ask the player questions. The user can click a green tick which means they think the question proposed is true, or they can click the red X to say they think its false.

The API collects 10 questions at a time, it is currently programmed to be fetching technology/programming related questions. When the player has completed the game, their score is presented to them.

We have a nice UI that I created using TKinter in python. This project was practice with making a basic UI, parsing JSON responses from a website to use the data within the JSON and manipulate it, we used the python requests module to retrieve the JSON, and then we used principles of OOP to seperate the different elements of our program.

Namely we had a quiz brain that ran the overall logic, we had a question model that modelled questions from the JSON data into formats easy for the quiz brain to use, and overall I found this project to be a great refresher on OOP principles and practicing abstraction.
