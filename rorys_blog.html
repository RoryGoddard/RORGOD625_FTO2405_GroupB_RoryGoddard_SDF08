This project was an exercise in a few technologies, namely Jinja, get and post requests, and Bootstrap for the CSS

I enjoyed this project, I was able to use Jinja to set up a base html file with html that would be common across all pages of a website, ie your header, footer, and whatever else wouldn't change between different pages.

From there, I could use python to dynamically insert the content into the HTML with Jinja, depending on the page I was on. So if I opened a blog post about cacti, then the blog title, paragraphs, and the image would load, but the rest of the content would remain the same

I was also able to submit new blog posts using post requests, which was nice practice and expore to REST API principles.

I really enjoyed the different aspects of this project, namely how easy navigating through different blog posts could be, because using jinja to pass arguments through to a dictionary in python, I could get the relevant information to change the elements on the page to match those of the requested blog post.
